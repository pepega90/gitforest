# gitforest

⚡ Interactive terminal UI for exploring Git history, browsing changed files, and viewing diffs directly from your terminal.

Built with Go + tview.

## Features

- 🌳 Commit tree explorer
- 📂 Expandable folder structure
- 🔍 Live commit search
- 🎨 Colored Git diffs
- ⚡ Fast terminal navigation
- 🖥 Pure terminal UI
- ⌨ Keyboard driven
- 🔎 File-level diff preview

---

## Installation

### Go Install

```bash
go install github.com/ajimustofa/gitforest
```

### Clone & Build

```bash
git clone https://github.com/ajimustofa/gitforest.git

cd gitforest

go build -o gitforest
```

---

## Usage

Run inside any Git repository:

```bash
gitpeek
```

---

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `↑ ↓` | Navigate |
| `Enter` | Expand / Select |
| `Tab` | Switch focus |
| `Ctrl+C` | Exit |
